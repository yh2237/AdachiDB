API ドキュメント

Web埋め込み

自分のサイトにランダム足立語録を埋め込むことができます。
以下のHTMLをコピーして、埋め込みたい場所に貼り付けてください。

埋め込みコード
HTML 
<div id="AdachiDB-random"></div>
<button id="AdachiDB-button">再読み込み</button>
<script src="https://adachi.2237yh.net/embed/random.js"></script>
GET /api/posts/random

ランダムな足立レイの投稿を1件取得します。

リクエスト例
bash
curl "https://adachi.2237yh.net/api/posts/random"
レスポンス例
JSON
読み込み中...
GET /api/posts/random10

ランダムな足立レイの投稿を10件取得します。

リクエスト例
bash
curl "https://adachi.2237yh.net/api/posts/random10"
レスポンス例
JSON
読み込み中...
GET /api/posts/search

ツイート本文を検索し、一致するツイートを返します。

パラメータ
パラメータ 必須 説明
q 必須 検索キーワード
limit 任意 取得件数（デフォルト: 100、最大: 40000）
from 任意 検索開始日（例: 2024-01-01）
to 任意 検索終了日（例: 2024-12-31）
リクエスト例
bash
curl "https://adachi.2237yh.net/api/posts/search?q=おはよう&limit=10"
レスポンス例
JSON
読み込み中...
GET /api/posts/id

指定したIDのツイートを取得します。

パラメータ
パラメータ 必須 説明
id 必須 データベース上のツイートID（数値）
リクエスト例
bash
curl "https://adachi.2237yh.net/api/posts/id?id=1"
レスポンス例
JSON
読み込み中...
GET /api/posts/all

全ツイートを新しい順に取得します。

パラメータ
パラメータ 必須 説明
limit 任意 取得件数（デフォルト: 200、最大: 40000）
リクエスト例
bash
curl "https://adachi.2237yh.net/api/posts/all?limit=50"
