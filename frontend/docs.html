- APIドキュメント

足立データベースは誰でも使用可能なAPIを公開しています。


- ランダムな足立レイの投稿を1件取得する

bash 
curl -X GET "https://adachi.2237yh.net/api/posts/random"

- ツイート本文を検索し、一致するツイートを返します。

bash 
curl -X GET "https://adachi.2237yh.net/api/posts/search?q={query}&limit={limit}"
  • {query}：検索したいキーワードを指定します。（必須）
  • {limit}：取得したい件数を指定します。最大2000件まで指定可能。デフォルトは100。（任意）
